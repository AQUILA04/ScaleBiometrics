# Story 1.1: Project Scaffolding & Monorepo Setup

**Status:** Draft

## Story
**As a** Lead Developer,
**I want** to initialize the monorepo structure and local infrastructure stack,
**so that** the team can immediately start implementing backend and frontend features in a consistent environment.

## Acceptance Criteria
1. **Monorepo Structure**: Root initialized with `apps/api`, `apps/web`, `packages/shared`, `infrastructure`.
2. **Backend Scaffold**: Spring Boot 3.2+ (Java 21) in `apps/api`.
3. **Frontend Scaffold**: Next.js 14+ in `apps/web`.
4. **Infrastructure**: `docker-compose.yml` with Postgres, Keycloak, Kafka, MinIO, Redis.
5. **Shared Config**: Root build configuration (Maven/Gradle/npm).
6. **Documentation**: `README.md` with startup instructions.

## Tasks
- [ ] Initialize Git & Folders.
- [ ] Create `infrastructure/local/docker-compose.yml`.
- [ ] Initialize Spring Boot (`mvn`).
- [ ] Initialize Next.js (`npx create-next-app`).
- [ ] Verify `docker-compose up` works.

## Dev Notes
* Java 21 required.
* Ports: Web(3000), API(8081), PG(5432), KC(8080/8180), MinIO(9000).
